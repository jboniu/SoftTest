# 项目要求
## 1.顶点类Vertex、圆形类Circle、多边形接口Polygon、三角形类Triangle、矩形类Rectangle、圆形适配类AdapterCircle按照图中关系图标构成适配器设计模式。
## 2.规范编写JavaDoc定义的文档注释，文档注释标签应包括对源码作者、编码版本或日期、传入参数、返回结果计算过程和类型等信息的介绍，为后续JavaDoc生成API说明文档做好准备工作。
## 3.设计测试用例，对数值计算方法进行单元功能测试，给出断言评测结果；设计测试用例，按照类图定义的关系检测实现类之间设计结构，利用反射机制确认各个类之间是否完整定义了继承、实现、方法重写等关系，并对各项一致性检测进行记录。
